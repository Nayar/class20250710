Git Commands

  • git status

    tells us which files have been changed and which branch we are on

  • git init

    Creates a new .git folder in the repository

  • git add {filename}

    stages a file, i.e. adds the changes of the file in a box

  • git commit -m '{commit message}'

    creates a new commit i.e. seals the box and put a message on it.

  • git push {remotename} {branchname}

    sends boxes to remote location

  • git pull {remotename} {branchname}

    sends boxes to remote location
